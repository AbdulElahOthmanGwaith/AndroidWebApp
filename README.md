# Android Web App Project

هذا المشروع عبارة عن تطبيق أندرويد متكامل مصمم لعرض مواقع الويب باستخدام تقنية **WebView** مع دعم كامل للميزات الحديثة.

## المميزات
- مبني باستخدام **Jetpack Compose**.
- دعم الروابط العميقة (Deep Linking).
- مراقبة حالة الشبكة (Network Monitoring).
- شاشة ترحيب (Splash Screen) مخصصة.
- دعم الإشعارات (Notifications).

## هيكل المشروع
- `app/src/main/java`: يحتوي على الكود المصدري بلغة Kotlin.
- `app/src/main/res`: يحتوي على الموارد (الصور، النصوص، الثيمات).

## كيفية التشغيل
1. قم بتحميل المشروع أو عمل Clone للمستودع.
2. افتح المشروع باستخدام **Android Studio**.
3. انتظر حتى ينتهي Gradle من تحميل التبعيات.
4. قم بتشغيل التطبيق على محاكي أو جهاز حقيقي.
